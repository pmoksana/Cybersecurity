## **Background / Scenario** 

Open Web Application Security Project (OWASP) is a non-profit organization that is dedicated to web application security. OWASP publishes a list of the Top 10 most critical web security concerns facing organizations about every three years. The objective of this report is to raise awareness regarding web application security and to help organizations incorporate this information into their process to minimize and mitigate security risks. 

OWASP also accepts community contributions for security related content. The OWASP Community Pages provides list of attacks and vulnerabilities reported by the communities. 

In this lab, you will explore the OWASP Top 10 from 2021 and a few of the attacks and vulnerabilities reported by community contributors.
